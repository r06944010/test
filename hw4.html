Assignment 4 (18%)

Assignment 4 has two parts. Details are shown below.

If there is any conflict or unclear things in the following, please follow the rules defined above.

Code (4%)

You are supposed to implement Generative Adversarial Network . Input a text and output images conditioning on the discription of the text.

Write one bash scripts named ‘run.sh' . TA will run your code by using command :
' bash run.sh [testing_text.txt] Make sure your code can be executed by this bash script with correct arguments setting.

Any other formats are not allowed and will not be graded. After finishing your work, write a report.pdf to describe how to run your code and analyze what you have done. At last, upload your code and report.pdf to github.
